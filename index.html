包含 MySQL Cluster NDB 7.3 参考手册

简介

这份文档是MySQL™参考手册。这个主要用于MySQL 5.6版(截止到5.6.15版) , 同时包含着MySQL Cluster 7.3 NDBCLUSTER (版本号:5.6.11-ndb-7.3.3)。

MySQL 5.6 特性. 这份文档描述了MySQL 5.6所有版本包含的所有特性,但并不是所有的特性都授权给你使用的MySQL 5.6版本, 如果你对使用的MySQL 5.6版本的特性有任何疑问，可以参考你的MySQL 5.6版的授权协议或者直接联系Oracle的销售代表来咨询。

你可以去MySQL 5.6 Release Notes.来查看每个发布版本的详细更新内容。

你可以去Legal Notices查看相关法律信息。
